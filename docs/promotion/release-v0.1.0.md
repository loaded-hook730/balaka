# Balaka v0.1.0

Initial public release of Balaka.

## Highlights

- local-only TTS backend built with FastAPI
- separate static frontend for quick browser use
- voice design flow
- voice cloning flow
- metadata endpoint for language and voice options
- simple Python 3.13 setup

## Notes

- no remote inference fallback
- frontend is build-free by design
- local runtime is tuned for a single `.venv` workflow
